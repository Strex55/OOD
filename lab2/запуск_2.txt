
Что сделать на Windows:
- : MSVC + CMake + vcpkg с SFML 3
  1) PowerShell (Админ):
     - git clone https://github.com/microsoft/vcpkg C:\vcpkg
     - C:\vcpkg\bootstrap-vcpkg.bat
     - C:\vcpkg\vcpkg.exe install sfml
     - Проверить, что ставится версия 3.x (vcpkg list sfml)
  2) Конфигурация/сборка:
     - cmake -S C:\OOD\Rep\lab2 -B C:\OOD\Rep\lab2\build -DCMAKE_TOOLCHAIN_FILE=C:/vcpkg/scripts/buildsystems/vcpkg.cmake -DBUILD_VISUALIZATION=ON -A x64
     - cmake --build C:\OOD\Rep\lab2\build --config Release
  3) Запуск:
     - C:\OOD\Rep\lab2\build\Release\geom_app.exe --input C:\OOD\Rep\lab2\input.txt --output C:\OOD\Rep\lab2\output.txt
